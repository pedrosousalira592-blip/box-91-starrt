# 🚗 MULTIRASTRO BOX 91 — Site Oficial

> **Estética Automotiva e Lava-Rápido Self Service — Tucumã, Pará**

---

## 📌 Sobre o Projeto

Site profissional e moderno para a **MULTIRASTRO BOX 91**, substituindo o site anterior que apresentava erro crítico de WordPress. O novo site é 100% estático, rápido, responsivo e focado em conversão para WhatsApp.

**Nota Google:** ⭐ 5,0 (49 avaliações)

---

## ✅ Funcionalidades Implementadas

- **Hero Section** com CTA principal para WhatsApp e animações de entrada
- **Barra de benefícios** destacando diferenciais (qualidade, agilidade, confiança)
- **Seção de Serviços** com 6 cards (Lava-Rápido, Polimento, Cristalização, Limpeza Interna, Lavagem de Motos, Loja de Acessórios)
- **Seção Produtos Vonixx** com showcase de linha de produtos
- **Avaliações do Google** em carousel interativo (autoplay + touch/swipe + drag)
- **Seção Sobre** com informações da empresa e localização
- **Mapa do Google Maps** integrado com botão de rota
- **CTA Final** com botões para WhatsApp e Instagram
- **Footer completo** com links, horários e contatos
- **Botão flutuante do WhatsApp** com tooltip animado
- **Navbar responsiva** com hamburguer menu para mobile
- **Efeito scroll** na navbar (backdrop blur + transparência)
- **Animações fade-in** ao scroll (Intersection Observer)
- **Partículas animadas** no hero
- **Animação das barras** de avaliações
- **Contador animado** das estatísticas
- **Back to top** button
- **Active link** na navbar conforme seção visível
- **SEO básico** com meta tags e Open Graph
